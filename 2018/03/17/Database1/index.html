数据库基本操作

发表于 2018-03-17 | 分类于 数据库
字数统计: 371 | 阅读时长 ≈ 2

库操作

  1. 退出数据库：exit；或quit；
  2. create database 库名；例如：create database myClass；
  3. 显示所有数据库：show databases;（查看当前有几个数据库）
  4. 删除数据库：drop database 库名;例如：drop database myClass;
  5. 进入数据库：use 库名；例如：use myClass;屏幕提示：Database Changed;
  6. 查看当前使用的数据库：select database（）;（查看当前在哪个数据库下）
  7. 当前数据库包含的表信息：show tables;（查看当前数据库有几个表）

表操作

  1. 建表：create table 表名（名称1 类型1，名称2 类型2…），例如：
    create table student（stunumber int（10），name char（20），age int（10）...）
  2. 获取表结构：desc 表名；例如：desc student;
  3. drop 表名；例如：drop student ;
  4. 插入数据：insert into 表名 字段1，字段2…value（值1，值2…），例如：
    insert into student(id,name,age) values （150930066，‘钟’，19），（150930077，‘刘’，19）;
  5. 查询表中数据：格式：select 名称1 from 表名 where 条件；例如；
    select * from student where stunumber=150930066;在student表中查询学号为150930066的所有信息；
  6. 删除表中信息：格式：delete from 表名 where 条件；例如；
    delete from student where age=19;
  7. 修改表中数据：格式：update 表名 set 名称=新值 where 条件；例如：
    update student set age=20 where name=‘钟’;
  8. 在表中增加字段：格式：alter table 表名 add 名称 类型 其他；例如：
    alter table student add sex char（10） default‘0’;
  9. 更改表名：格式：rename table 原表名 to 新表名；：例如：把student改成myStudent；
    rename table student to myStudent;
