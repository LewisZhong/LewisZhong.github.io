正则表达式

正则函数

  1. preg_match($zz,$string,$matches) 分别为正则表达式，待匹配字符串，匹配到的字符串

原子

  1. /d 代表0-9中任意一个数字
  2. /w 代表任意一个数字或字母
  3. /s 代表任意一个空白字符
  4. [^a-d] 代表任意一个非a-d的字母

元字符

  1. * 0到多
  2. + 1到多
  3. ？ 0或1
  4. . 1个原子，但不包括换行符
  5. ^a 必须以a开始
  6. b$ 必须已b结尾
  7. /b 词边界
  8. {m} 只能出现m次
  9. {n，m} 出现n到m次
  10. {m，} 至少出现m次

模式修正符

  1. i $pattern = '/ABC/i'; 忽略大小写
  2. m $pattern = '/^a\d+/m'; 视为多行，适用于待匹配的字符串比较长有多行的情况
  3. s $pattern = '/新的未来.+\d+/s'; 此模式下.可匹配任意一个字符串，包括换行符
  4. x $pattern = '/a b c /x'; 忽略空白符
  5. U $pattern = '/<div>.*<\/div>/U';取消贪婪模式，即匹配到一次便放弃继续匹配
