PHP引用变量考察点

什么是引用变量？在PHP当中，用什么符号定义引用变量？

概念:在PHP中意味着用不同的名字访问同一个变量内容
定义方式:使用&符号
unset()只会取消引用，不会销毁空间
对象本身就是引用传递，如果想复制对象，可用clone方法

常量及数据类型考察点

PHP中字符串可以使用哪三种定义方法以及各自的区别是什么？

  1. 双引号：双引号可以解析变量，变量可以使用特殊字符和{}包含；双引号可以解析所有转义字符
  2. 单引号：单引号不能解析变量；单引号不能解析转义字符，只能解析单引号和反斜线本身；变量和变量，变量和字符串，字符串和字符串之间用.连接；单引号效率更高

  3. heredoc和newdoc：heredoc类似双引号，newdoc类型单引号,两者都用来处理大文本

数据类型

标量：整型、浮点型、字符串、布尔

  1. 浮点类型不能用于比较运算，因为浮点数在转换成二进制时有损耗，不精确
  2. bool值为false的七种情况：整型0、浮点0.0、布尔false、空字符串、0字符串、空数组、NULL
  3. 常量：

    const:const更快，是语言结构，可以用于类常量的定义
    define:define是函数，define不能用于类常量的定义

复合：数组、对象

  1. 超全局数组：$GLOBALS, $_GET, $_POST, $_SESSION, $_COOKIE, $_SERVER, $_FILES, $_REQUEST, $_ENV
  • 服务器端IP地址：$_SERVER['SERVER_ADDR']

流程控制考察点

PHP中三种数组的遍历方式以及它们的区别？

  1. for循环：只适用于索引数组

  2. foreach:可同时用于索引数组与关联数组,会将数组指针reset回到开头

  3. while-list-each:可同时用于索引数组与关联数组，会改变数组指针，不会reset

each函数用来取出数组中当前指针指向的key/value对，然后将指针后移一位
list函数用来将each函数取出的key/value对的值分别赋给两个变量

自定义函数及内部函数考察点

函数的引用返回

  1. 类/函数内部定义的变量为局部变量，外边的为全局变量，若想在类/函数内部调用全局变量，可在变量前加global关键字或使用$GLOBALs[]超全局数组
  2. 静态变量static仅在函数局部域中生效，但当程序执行离开此作用域时，其值并不会消失。静态变量仅初始化一次；初始化时需要赋值，否则为NULL；每次执行函数该值都会保留；static修饰的变量都是局部的，仅在函数内部有效；static可以记录函数的调用次数，可在某些条件下终止递归
  3. 默认情况下，函数参数通过值传递，如果希望允许函数修改外部变量的值，必须通过引用传递参数&

外部文件的导入

include():如果要导入的文件不存在，则会发出一个警告错误，脚本继续运行
include_once():会检查文件是否已经被包含过，如果是则不会再次包含
require():如果要导入的文件不存在，则会发出一个致命错误，脚本中止运行
require_one():会检查文件是否已经被包含过，如果是则不会再次包含

系统内置函数

1.日期时间函数

  • date_default_timezone_set — 设定用于一个脚本中所有日期时间函数的默认时区
  • date — 格式化一个本地时间／日期
  • strtotime — 将任何字符串的日期时间描述解析为 Unix 时间戳
  • mktime — 取得一个日期的 Unix 时间戳
  • time — 返回当前的 Unix 时间戳
  • microtime — 返回当前 Unix 时间戳和微秒数

2.IP处理函数

  • ip2long — 将 IPV4 的字符串互联网协议转换成长整型数字
  • long2ip — 将长整型转化为字符串形式带点的互联网标准格式地址（IPV4）

3.打印处理

  • print — 是语言结构而不是函数，只可以输出一个变量，并且返回1
  • echo — 是语言结构而不是函数，可以输出一个或多个变量，没有返回值
  • printf — 输出格式化字符串
  • print_r — 打印关于变量的易于理解的信息
  • sprintf — 返回一个格式化的字符串
  • var_dump — 打印变量的相关信息
  • var_export — 输出或返回一个变量的字符串表示

4.序列化及反序列化函数

  • serialize() 返回字符串，此字符串包含了表示 value 的字节流，可以存储于任何地方。
  • unserialize() 对单一的已序列化的变量进行操作，将其转换回 PHP 的值

5.字符串处理函数

  • implode — 将一个一维数组的值转化为字符串
  • explode — 将一个字符串分割为数组
  • strrev — 反转字符串
  • strstr — 查找字符串的首次出现
  • trim — 去除字符串首尾处的空白字符（或者其他字符）
  • number_format — 以千位分隔符方式格式化一个数字

6.数组处理函数

  • array_keys — 返回数组中部分的或所有的键名
  • array_values — 返回数组中所有的值
  • array_diff — 计算数组的差集
  • array_intersect — 计算数组的交集
  • array_merge — 合并一个或多个数组
  • array_shift — 将数组开头的单元移出数组
  • array_unshift — 将数组末尾的单元移出数组
  • array_pop — 弹出数组最后一个单元（出栈）
  • array_push — 将一个或多个单元压入数组的末尾（入栈）
  • sort — 对数组排序

正则表达式考察点

1.正则表达式的作用

分割、查找、匹配、替换字符串

2.分隔符

正斜线(/),hash符号(#)以及取反符号(~)

3.通用原子

\d:0-9, \D:非0-9,\w:数字字母下划线,\W；非数字字母下划线,\s：空白符,\S：除非空白符

4.元字符

  • .：除了换行符之外的任意字符
  • *：0或多
  • ?：0或1
  • ^：必须以它开头
  • $：必须以它结尾
  • +：1或多
  • {n}：出现n次
  • {n,m}：出现n-m次
  • {n,}：出现n次以上
  • []：集合，匹配[]中的一个
  • ()：代表整体
  • [^]：取反
  • |：或
  • [-]：范围，如0-9

5.模式修正符

  • i $pattern = '/ABC/i'; 忽略大小写
  • m $pattern = '/^a\d+/m'; 视为多行，适用于待匹配的字符串比较长有多行的情况
  • s $pattern = '/新的未来.+\d+/s'; 此模式下.可匹配任意一个字符串，包括换行符
  • x $pattern = '/a b c /x'; 忽略空白符
  • U $pattern = '/<div>.*<\/div>/U';取消贪婪模式，即匹配到一次便放弃继续匹配
