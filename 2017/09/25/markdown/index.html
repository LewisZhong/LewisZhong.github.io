    列表

    无序列表

    • 用 *或+或- 来做前导
    • 符号后面要记得添加一个空格
    • 否则不会是无序列表
    • 而是段落

    有序列表

    1. 用数字加英文的句号来做前导
    2. 只要是数字就行
    3. 但为了看源码方便
    4. 推荐用1,2,3,4…

    代码区

    • markdown兼容了HTML的所有语法
    • 缩进四个空格或一个制表符
    • 或者用标签对
      echo “hello world”
    • 或者hello world

    分隔线

    用三个以上星号、减号、底线来建立一个分隔线，行内不能有其他东西
